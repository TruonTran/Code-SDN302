## ✅ Task 3: API – POST /users/register


POST "http://localhost:8386/users/register"


### Postman Setup:
- **Method:** POST
- **Tab:** Body → raw → JSON
- **Headers:** Content-Type: application/json (auto-set)

json

    "name": "Test student",
    "email": "teststudent@course.com",
    "password": "password123"

## ✅ Task 4: API – POST /users/login

**Objective:** Authenticate user and receive JWT token

### Endpoint:

POST http://localhost:8386/users/login

json
{
    "email": "teststudent@course.com",
    "password": "password123"
}

### ⚠️ IMPORTANT:
**Save the token value from the response!** You will need it for Task 5.


## ✅ Task 5: API – POST /enrollments

**Objective:** only users with the role "student" can enroll in a course using JWT authentication

### Endpoint:

POST http://localhost:8386/enrollments

### Postman Setup - Method 2 (Authorization Tab):
- **Method:** POST
- **Tab:** Body → raw → JSON
- **Tab:** Authorization
  - Type: `Bearer Token`
  - Token: `<paste_token_from_task_4>` (without "Bearer" prefix)

### Request Body (Copy-Paste):
json
{
  "courseId": "65f100000000000000000001"
}
